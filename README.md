## Install
* Download the `kraken.zsh-theme` file.
* Move kraken.zsh-theme file to oh-my-zsh's custom themes folder: `~/.oh-my-zsh/custom/themes/`.
* Change `ZSH_THEME="kraken"` the in `~/.zshrc`.
